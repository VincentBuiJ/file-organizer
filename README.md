# File Organizer
- It automatically organizes folders by the file type.

## What It Does:
- Scans a directory for files
- Group files by their extension (.txt, .pdf, .py)
- Creates organized folders by type
- Moves files into appropriate folders

## How to Use
- python file_organizer.py

## Future Improvements
- Let user choose which folder to organize
- Add undo functionality
